## üí° 1. Funciones

### ‚û§ ¬øQu√© es una funci√≥n?

Una **funci√≥n** es un bloque de c√≥digo que realiza una tarea espec√≠fica.
Permite dividir un programa en partes m√°s peque√±as, reutilizables y legibles.

```python
def saludar(nombre):
    print(f"Hola, {nombre}!")

saludar("Luc√≠a")
```

**Salida:**

```
Hola, Luc√≠a!
```

---

### ‚û§ Retorno de valores

Las funciones pueden devolver resultados con `return`.

```python
def sumar(a, b):
    return a + b

resultado = sumar(3, 5)
print("Resultado:", resultado)
```

---

### ‚û§ Tipos de argumentos

| Tipo         | Ejemplo            | Descripci√≥n                                    |
| ------------ | ------------------ | ---------------------------------------------- |
| Posicionales | `sumar(2, 3)`      | Se pasan en orden.                             |
| Nombrados    | `sumar(a=2, b=3)`  | Se especifica el nombre del par√°metro.         |
| `*args`      | `def f(*args):`    | Recibe una tupla de n argumentos.              |
| `**kwargs`   | `def f(**kwargs):` | Recibe un diccionario de argumentos nombrados. |

#### Ejemplo:

```python
def mostrar_info(*args, **kwargs):
    print("Argumentos posicionales:", args)
    print("Argumentos nombrados:", kwargs)

mostrar_info(1, 2, 3, nombre="Ana", edad=22)
```

**Salida:**

```
Argumentos posicionales: (1, 2, 3)
Argumentos nombrados: {'nombre': 'Ana', 'edad': 22}
```

---

### ‚û§ Funciones sin retorno y funciones que mutan el estado

Algunas funciones **no devuelven valores** (solo realizan una acci√≥n):

```python
def mostrar_mensaje():
    print("Ejecuci√≥n completada.")
```

Otras **modifican** datos externos (mutan el estado):

```python
def agregar_elemento(lista, valor):
    lista.append(valor)

valores = [1, 2]
agregar_elemento(valores, 3)
print(valores)  # [1, 2, 3]
```

En cambio, las funciones **puras** no alteran variables fuera de su √°mbito:

```python
def nueva_lista(lista, valor):
    return lista + [valor]
```

üß© *Introducci√≥n breve:* este concepto es la base de las **m√°quinas de estado**, donde el sistema cambia de situaci√≥n (estado) seg√∫n las acciones o entradas recibidas.

---

### ‚û§ Funciones anidadas y lambda

Las **funciones anidadas** son aquellas definidas dentro de otra funci√≥n. Son √∫tiles para encapsular l√≥gica interna o comportamientos auxiliares.

```python
def elevar(base, exponente):
    def potencia(b, e):
        return b ** e
    return potencia(base, exponente)

print(elevar(2, 3))  # 8
```

---

Las **funciones lambda** son funciones an√≥nimas (sin nombre), de una sola l√≠nea y con retorno impl√≠cito.
Su sintaxis general es:

```python
lambda par√°metros: expresi√≥n
```

Se utilizan com√∫nmente para operaciones simples o como argumentos de otras funciones (por ejemplo, en `map()`, `filter()` o `sorted()`).

#### Ejemplos b√°sicos:

```python
# Ejemplo 1: funci√≥n lambda simple
doble = lambda x: x * 2
print(doble(5))  # 10

# Ejemplo 2: suma de dos valores
sumar = lambda a, b: a + b
print(sumar(3, 7))  # 10

# Ejemplo 3: filtrar n√∫meros pares en una lista
numeros = [1, 2, 3, 4, 5, 6]
pares = list(filter(lambda x: x % 2 == 0, numeros))
print(pares)  # [2, 4, 6]
```

---

### üß† Nota:

> Las funciones deben tener **nombres descriptivos** (`snake_case`) y se recomienda usar **docstrings** para documentar su uso.
> Las funciones lambda deben usarse **solo cuando la operaci√≥n sea corta y clara**; para l√≥gica compleja, es mejor definir una funci√≥n tradicional.

## ‚öôÔ∏è 2. Manejo de Excepciones

Las **excepciones** permiten manejar errores sin interrumpir la ejecuci√≥n del programa.

```python
try:
    x = int(input("Ingrese un n√∫mero: "))
    print(10 / x)
except ZeroDivisionError:
    print("No se puede dividir por cero.")
except ValueError:
    print("Debe ingresar un n√∫mero v√°lido.")
finally:
    print("Fin del programa.")
```

### üß† Nota:

> Validar los datos antes de procesarlos es una buena pr√°ctica. Las excepciones deben usarse para casos no esperados.

## üì¶ 3. M√≥dulos y Paquetes

### ‚û§ M√≥dulos

Un **m√≥dulo** es un archivo `.py` que contiene c√≥digo reutilizable (funciones, clases o variables).

```python
# archivo: operaciones.py
def sumar(a, b):
    return a + b
```

Luego puede importarse desde otro archivo:

```python
import operaciones
print(operaciones.sumar(2, 3))
```

Tambi√©n se puede importar una funci√≥n espec√≠fica:

```python
from operaciones import sumar
print(sumar(5, 8))
```

Y usar alias:

```python
import operaciones as op
print(op.sumar(1, 2))
```

---

### ‚û§ Creaci√≥n de paquetes

Un **paquete** es una carpeta que agrupa varios m√≥dulos. Debe contener un archivo `__init__.py` (aunque est√© vac√≠o):

```
mi_paquete/
‚îú‚îÄ‚îÄ __init__.py
‚îú‚îÄ‚îÄ matematicas.py
‚îî‚îÄ‚îÄ estadistica.py
```

#### Ejemplo:

```python
# matematicas.py
def promedio(valores):
    return sum(valores) / len(valores)
```

```python
# main.py
from mi_paquete.matematicas import promedio
print(promedio([2, 4, 6]))  # 4.0
```

üß† *Importaciones relativas:* permiten referirse a otros m√≥dulos del mismo paquete usando `from .modulo import funcion`.

## üß∞ 4. Librer√≠as est√°ndar de Python

Python incluye un gran conjunto de m√≥dulos preinstalados llamados **librer√≠as est√°ndar**.

### Ejemplos comunes:

| M√≥dulo     | Uso principal             | Ejemplo                 |
| ---------- | ------------------------- | ----------------------- |
| `math`     | Funciones matem√°ticas     | `math.sqrt(9)` ‚Üí 3.0    |
| `os`       | Operaciones del sistema   | `os.listdir('.')`       |
| `datetime` | Fechas y horas            | `datetime.date.today()` |
| `random`   | N√∫meros aleatorios        | `random.randint(1, 10)` |
| `sys`      | Par√°metros del int√©rprete | `sys.version`           |

#### Ejemplo pr√°ctico:

```python
import os, math, datetime, random

print("Archivos en el directorio actual:", os.listdir('.'))
print("Ra√≠z cuadrada de 16:", math.sqrt(16))
print("Fecha actual:", datetime.date.today())
print("N√∫mero aleatorio:", random.randint(1, 5))
```
